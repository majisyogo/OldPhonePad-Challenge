# Old Phone Pad - C# Coding Challenge (Iron Software)

This project was created as part of the **AI Software Engineer (Junior)** coding challenge from Iron Software.

It simulates the behavior of an old mobile phone keypad:
users can input numbers, and the program converts them into letters
according to traditional T9-style mapping.
